# Gestor de Movimientos 

<img width="803" height="481" alt="imagen" src="https://github.com/user-attachments/assets/be3413e5-c8a8-41c4-9187-8fd859119833" />

Proyecto realizado para la asignatura de Acceso a Datos. Consiste en una aplicación simple de CRUD en WinForms conectada a una base de datos local. 

Ventanas de la aplicación: 
- Ventana principal en la que puedes ver tus últimos 5 movimientos y varios botones uno para actualizar la tabla de movimiento y otros para ir a las siguientes ventanas.
- Ventana para agregar nuevos movimientos de dos tipos: ingreso y gasto.
- Ventana para eliminar movimientos seleccionándolos en la tabla. 
-  Ventana para modificar la descripción de un movimiento. 
- Ventana para ver todos los movimientos empezando por los más recientes y poder filtrarlos por tipo.
